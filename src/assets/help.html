Sökning

I Minoritetsordlistan kan du söka på svenskt ord och få den finska översättningen. Du kan också byta sökriktning och söka på finskt ord för att få den svenska motsvarigheten. Det gör du genom att klicka på pilarna ().

Innan några träffar visas så måste du välja ett sakområde. Det går att söka i flera sakområden samtidigt.

Du kan välja mellan att söka på Ord börjar med eller  Ord innehåller.  Ord börjar med tar fram alla ord som börjar med den bokstavssekvens du skriver in. Ord innehåller ger alla ord som innehåller söksträngen någonstans alls (exempelvis ger sökning på ’arv’ även ’läkarvård’).

Sökningen börjar så fort du skriver något, men du kan alltid fortsätta skriva för att få ett snävare resultat.

Vissa ord har underuppslag som inte visas direkt. Dem når du genom att klicka på pilen till höger ().

Sökningen ger som mest tillbaka 500 sökträffar, detta med tanke på mobilandvändning vid dålig mottagningen. För att komma åt alla träffar kan du använda HTML-exporten (se nedan).

Bokmärkning

Alla inställningar sparas i sidans webbadress (URL). Om du t.ex. väljer att söka på finska istället för svenska via pilarna i språkväljarnen så läggs searchLang=fin till. Om du sökt på något visst ord så finns det också med i URL:en.

Det innebär att det är bäst att ha de inställningar man vill och vara på rätt nivå när man skapar ett bokmärke för tjänsten – dvs välja det sökspråk, söktyp, och ev. återkommande sakområden först, och sedan spara ett bokmärke i din webbläsare. Då får du de inställningarna varje gång du går in via bokmärket.

Tips för att lägga till bokmärke på hemskärm i mobilen

I din mobil rekommenderar vi att du lägger bokmärket på din hemskärm. Det kommer då att se ut som de andra apparna, men öppnas i webbläsaren.

Android, Firefox: Öppna menyn, välj Page och sedan Add Page Shortcut.

Android, Chrome: Öppna menyn, välj Add to homescreen.

Även söksträngen sparas, så vi rekommenderar att du inte skriver in en söksträng innan du skapar bokmärket. Däremot är det praktiskt att kunna dela länkar till en viss sökning.

Export

Via HTML-symbolen nertill () kan du skapa en HTML-export av sökningen. Den öppnas i din browser direkt och kan laddas ner. Du kan också välja att skriva ut sökresultaten eller spara som pdf via webbläsarens funktioner.
